# Contributing to Sanovia Landing Page

Nous accueillons les contributions à notre landing page Sanovia ! Voici comment vous pouvez contribuer.

## 🚀 Comment contribuer

### 1. Fork le projet
- Cliquez sur le bouton "Fork" en haut de la page
- Clonez votre fork localement

### 2. Créez une branche
```bash
git checkout -b feature/ma-nouvelle-fonctionnalite
```

### 3. Effectuez vos modifications
- Assurez-vous que votre code suit nos standards
- Testez vos modifications sur différents appareils
- Vérifiez la compatibilité mobile

### 4. Committez vos changements
```bash
git commit -m "Add: Description claire de votre ajout"
```

### 5. Poussez vers votre fork
```bash
git push origin feature/ma-nouvelle-fonctionnalite
```

### 6. Créez une Pull Request
- Décrivez clairement vos modifications
- Ajoutez des captures d'écran si applicable

## 📋 Guidelines

### Code Style
- Utilisez des noms de variables explicites
- Commentez le code complexe
- Respectez l'indentation existante
- Suivez les conventions CSS BEM si applicable

### Responsive Design
- Testez sur mobile, tablette et desktop
- Utilisez les breakpoints existants
- Vérifiez l'accessibilité

### Performance
- Optimisez les images
- Minimisez les animations coûteuses
- Testez les temps de chargement

## 🐛 Signaler un bug

1. Vérifiez qu'il n'existe pas déjà
2. Utilisez le template d'issue
3. Incluez les étapes de reproduction
4. Ajoutez des captures d'écran

## 💡 Suggestions

- Utilisez les discussions GitHub
- Proposez des améliorations UX/UI
- Partagez vos idées d'optimisation

## 📞 Contact

Pour toute question, contactez l'équipe Sanovia.

Merci de contribuer à Sanovia ! 🏥
